This page presents a detailed case study of the 94/3 Online Shopping Store, featuring insights into the Project Overview, Tools Used, and direct Live Links to the official products.

Project Overview

"94/3 is a responsive full-stack web app, conceived as part of 'The Odin Project'. This online shopping store was an opportunity for me to showcase my frontend skills and reinforce my foundations in React.js. Through the use of React.js, React Router, and CSS, I crafted a seamless and responsive user interface compatible with all devices.

Authentication is handled through Firebase, ensuring a secure user experience, while Stripe facilitates trustworthy payment processing. This project not only fulfilled a task requirement but also provided valuable hands-on experience in frontend development.
