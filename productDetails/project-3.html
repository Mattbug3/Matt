WM

This page presents a detailed case study of the WM online shopping store, featuring insights into the Project Overview, Tools Used, and direct Live Links to the official products.

Project Overview

WM is a fully responsive pure front-end project developed using React.js, React Router, CSS, and enhanced with SwipeJS for a dynamic touch. The web app encompasses a homepage, shopping page, blog page, contact page, about us page, and a login/register page. Users can effortlessly explore and select clothes based on their gender, size, and various genres.

This project was conceived after completing the 94/3 online shopping store. While 94/3 demonstrates a responsive full-stack approach, WM focuses on elevating the front-end experience with enhanced interactivity and aesthetics, including the seamless touch interactions brought by SwipeJS.
